CARDIAC ORGANOID ANALYSIS

MyoOptix

Desktop software for quantitative analysis of cardiac organoid contractility — measuring heart rate, beat intervals, contraction and relaxation dynamics from brightfield video.

🎯

No calibration needed

Automatic ROI detection via U-Net or Otsu segmentation.

⚡

Quick Analysis

Single-video analysis with the full MyoOptix pipeline.

📊

Full metrics

HR, IBI, CS/CE/RE, contractility — same as the main app.

🖥️

Runs locally

Native Mac and Windows app — video never leaves your machine.
